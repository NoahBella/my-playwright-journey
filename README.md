# 🎭 My Playwright Automation Journey

## 🚀 About This Project
I am a **Manual Tester** transitioning into **Automation Engineering**. This repository tracks my 30-day journey learning Playwright from scratch.

## 🛠️ What I've Learned So Far
* **Week 1:** Installation, Locators, Regex basics, and using Codegen to record scripts.
* **Week 2:** Version Control with Git/GitHub (Current).

## 🏃 How to Run the Tests
1. Clone the repo: `git clone https://github.com/NoahBella/my-playwright-journey.git`
2. Install dependencies: `npm install`
3. Run tests: `npx playwright test`
